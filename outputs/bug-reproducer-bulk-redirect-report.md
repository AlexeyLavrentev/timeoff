## Root cause

The nested bulk routes copied the relative redirect used by one-level single-request routes.

## Approved fix

Changed all three bulk-handler outcomes to use the absolute /requests/ redirect.

**Why this is causal:** The explicit target cannot resolve relative to the extra /bulk/ URL segment.
